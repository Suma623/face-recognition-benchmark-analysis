# Face Recognition Benchmark Analysis

A comparative study of PCA, LBPH, and FaceNet models across multiple benchmark datasets and image preprocessing techniques.

## Key Highlights
- Compared 3 face recognition models
- Evaluated 54 model-dataset-preprocessing combinations
- Used AT&T, FEI, and Faces94 datasets
- Achieved 100% accuracy using FaceNet on the AT&T dataset
- Evaluated using Accuracy, Precision, Recall, F1-Score, PSNR, and SSIM

## Technologies
Python, OpenCV, Scikit-Learn, FaceNet, Computer Vision, Digital Image Processing
